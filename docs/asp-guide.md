## ASP登録手順

### A8.net
1. https://www.a8.net/ にアクセス
2. 「無料登録」→ 必要情報入力
3. メディア情報として「SNSアカウント（Threads/X）」を登録
4. 審査通過後、「電話占い」で検索
5. ヴェルニ・ココナラ等の案件に提携申請
6. 承認後、アフィリエイトリンクを `knowledge/uranai/asp-links.json` に設定

### 注意事項
- SNSアフィリエイトが許可されている案件のみ使用
- PR表記（※PR / ※広告）を必ず付ける
- 1日1回まで（スパム認定回避）
- 同一案件は3日以上間隔を空ける
